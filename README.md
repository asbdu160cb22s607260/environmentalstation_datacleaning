# Environmental Data Cleaning Project

This project focuses on cleaning and preparing environmental station
data for analysis. Using Python, Pandas, and YData Profiling, the raw
dataset is processed to remove errors, handle missing values, fix
formatting issues, and generate a profiling report.

## Files in this Project

-   **Environmental_Station.csv** -- Raw dataset\
-   **EnvironmentalStation_Cleaned.csv** -- Cleaned dataset\
-   **environmental_station.ipynb** -- Data cleaning code\
-   **report.html** -- YData Profiling report\
-   **README.md** -- Project documentation

## Steps Performed

1.  Loaded raw data\
2.  Handled missing values\
3.  Cleaned and formatted columns\
4.  Removed duplicates and outliers\
5.  Generated profiling report\
6.  Saved cleaned dataset

## Tools Used

-   Python\
-   Pandas\
-   Jupyter Notebook\
-   YData Profiling
